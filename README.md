# react-event

Starting repo for React hackaton

Production version is available here:
react-event-delta.vercel.app

## Getting started

> [!TIP]
> You must use NodeJS >= 18

```bash
git clone git@github.com:mbredikhin/react-event.git
cd react-event
npm install
npm run dev
```

## Production build

```
npm run build
```
